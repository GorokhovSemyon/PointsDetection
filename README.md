# PointsDetection

## About

- One of the possible implementations of finding points in a video sequence by methods provided by OpenCV.
- 0 - SIFT
- 1 - SURF
- 2 -- BRISK

## Run

> $ git clone https://github.com/GorokhovSemyon/PointsDetection.git

> $ cd PointsDetection

> $ mkdir build

> $ cd build

> $ cmake ..

> $ make

> $ ./PointDetect d=<0 or 1 or 2>
